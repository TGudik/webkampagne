Skab noget virkeligt med kodning

På uddannelsen arbejder du både med grafisk design, brugergrænseflader (UI) og kodning. Det betyder, at du lærer at tage en idé og udvikle den til en rigtig hjemmeside eller digital løsning. Du arbejder med både det kreative og det tekniske, så du kan skabe noget, der både ser godt ud og fungerer i praksis. Undervisningen er praktisk orienteret, så du løbende kan se dine projekter blive til virkelighed.

Varighed og den gode løn

Uddannelsen til webudvikler tager typisk 2-3 år, afhængigt af hvilken vej man vælger.

Undervejs kan man få praktisk erfaring gennem praktik i en virksomhed.

Webudviklere er efterspurgte, og lønnen er derfor god.

om måneden, med mulighed for endnu højere løn med erfaring.

Social støtte og et trygt studiemiljø

På uddannelsen lægges der vægt på et godt socialt miljø, hvor du kan få støtte fra både undervisere og medstuderende. Mange opgaver løses i samarbejde, så man kan hjælpe hinanden og dele idéer. Hvis du har brug for ekstra støtte i din uddannelse, er der også mulighed for vejledning og hjælp, så du kan gennemføre uddannelsen på en måde, der fungerer for dig.

Se hvad du lærer

Herunder er der tre kort der fortæller hvad de primære kodesprog du skal lære på uddannelsen handler om. Herudover, kan du toggle på knapperne og se hvad de forskellige sprog gør i praksis.

HTML

HTML (HyperText Markup Language) er grundlaget for en hjemmeside og bruges til at opbygge strukturen på en webside, f.eks. overskrifter, tekst, billeder og links.

HTML

CSS

CSS (Cascading Style Sheets) bruges til at style hjemmesiden. Det bestemmer blandt andet farver, skrifttyper, layout og hvordan siden ser ud på forskellige skærme.

CSS

JAVASCRIPT

JavaScript (JS) bruges til at gøre hjemmesider interaktive. Det kan f.eks. håndtere klik på knapper, vise eller skjule indhold og opdatere siden uden at genindlæse den.
